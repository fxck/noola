A single cloud dev container with the whole Noola monorepo checked out and idling, plus
PostgreSQL, Valkey, NATS, Typesense, Qdrant, object storage, a shared embedder and
Mailpit. Connect your IDE or SSH in and run the api and web dev servers (both Node)
right next to the data. The Elixir edge (realtime) and the docker flow runner aren't in
this single Node box — use the AI Agent rung for the full stack. Great for editing Noola
from a thin client or a fresh machine without a local toolchain.
